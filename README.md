### 仿照网易云音乐小程序

小程序主要用原生框架开发，用到微信开发者工具自带的一个云函数，通过云函数来操作服务器，不用自己购买服务就可以实现外网访问，云函数是最近比较火的概念，专有名称叫做Serverless，基于node.js的全栈开发，无后端，前端专注于业务

### 启动
  把文件  git clone 下来 ，放到微信开发者工具直接运行就可以了
